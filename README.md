# 体检预约后台管理系统 java607

## 项目概述

体检预约后台管理系统是基于前后端分离架构开发，采用 Vue.js 与 Node.js 技术栈，实现了体检预约流程的全面管理。

## 技术栈

- 开发工具：VS Code、Navicat Premium
- 前端：Vue、ElementUI
- 后端：Node.js
- 数据库：MySQL 8.2.0
- Node版本：v16.20.2

## 功能模块

- 登录
- 系统管理
  - 项目管理
  - 套餐管理
  - 记录管理
- 日志管理
- 个人管理
- 轮播图管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 前端：支持现代浏览器，推荐使用 Chrome、Firefox
- 后端：Node.js 环境，MySQL 数据库

## 部署步骤

1. 安装 Node.js 环境及 MySQL 数据库
2. 导入数据库文件至 MySQL
3. 配置后端环境及数据库连接信息
4. 运行后端服务
5. 前端通过 npm 或 yarn 安装依赖
6. 前端项目配置代理或修改请求地址以匹配后端服务
7. 运行前端项目

## 目录结构

```
/
├── frontend                # 前端代码
└── backend                 # 后端代码
    ├── routes              # 路由
    ├── models              # 数据模型
    ├── controllers         # 控制器
    ├── middleware          # 中间件
    ├── utils               # 工具函数
    ├── config              # 配置文件
    └── app.js              # 入口文件
```

## 核心亮点

- 前后端分离，易于维护与扩展
- 采用 Vue + ElementUI，界面简洁直观
- 集成权限管理，确保系统安全可靠
